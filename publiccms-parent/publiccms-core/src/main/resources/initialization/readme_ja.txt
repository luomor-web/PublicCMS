表構造の変更とデータ調整
1. 全量スクリプトを修正する(sql/init.sql)
2. 増分スクリプトを追加する(sql/{fromVersion}-{toVersion}.sql)
増分スクリプト
1. タイムラインの記録を追加する(-- {変更日} --)
2. 修正内容を追加する(DML,DDL)[DMLはできるだけ1行に、DDLは項目ごとに1行]
データベースの手動初期化とアップグレード
1. データベースを初期化するには、sqlディレクトリ内のinit.sqlスクリプトを実行するだけで済む
2. 安定版のデータベースをアップグレードするには、({旧バージョン番号}-{新バージョン番号}.sql)を実行するだけで済む
3. プレビューバージョンのデータベースをアップグレードするには、時間軸に従って({最新バージョン番号}-develop.sql)内のソースコードのダウンロード日以降のアップグレードスクリプトを実行する必要がある
